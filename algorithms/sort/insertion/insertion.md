### INSERTION SORT ###
---------------------------

Insertion sort is an in-place memory swap sorting algorithm that runs in _o_(n^2) time.


|           | Worst    | Best   | Average  | Worst-Space |
| --------- | -------- | ------ | -------- | ----------- |
| Insertion | _O_(n^2) | _O_(n) | _O_(n^2) | _O_(n)      |
